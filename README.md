## Тематика сайта, название
**Wallpaper Vault** — интерактивная галерея обоев для различных устройств. Идея реализовать современный веб-каталог высококачественных изображений для рабочего стола (Desktop) и мобильных платформ (Android/iOS), включая поддержку анимированных фонов для Wallpaper Engine.

## Цель и задачи сайта
Обеспечение быстрого и удобного доступа к графическому контенту для персонализации цифрового пространства пользователя.
* реализация динамической фильтрации контента по категориям (Минимализм, Космос, Природа, iOS/Tendies);
* верстка адаптивной сетки для корректного отображения превью как для широкоформатных мониторов, так и для вертикальных экранов смартфонов;
* разработка интерфейса модального окна для просмотра оригинальных изображений и превью живых обоев.

## Целевая аудитория
Пользователи персональных компьютеров и мобильных устройств, дизайнеры и геймеры. Целевая аудитория включает владельцев iOS (ценителей стиля tendies) и пользователей Wallpaper Engine, что требует особого внимания к форматам изображений и качеству визуальных эффектов.

## Содержание сайта
* шапка сайта с расширенным меню категорий (Desktop, Phone, Live);
* основная галерея с адаптивной сеткой карточек изображений разного формата;
* модальное окно для детального просмотра и выбора разрешения под конкретное устройство;
* подвал сайта с информацией об источниках и ссылками на сообщества.

## Функциональные возможности
* мгновенная фильтрация изображений по тематическим тегам;
* просмотр изображений в полном разрешении и поддержка воспроизведения видео-превью для Wallpaper Engine;
* анимированные эффекты перехода, вдохновленные стилистикой iOS;
* полная адаптивность сайта под мобильные и десктопные разрешения.
