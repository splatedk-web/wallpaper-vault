# Итоговый проект по дисциплине Веб-технологии

## ФИО студента, номер учебной группы
Пономарев Даниил, 2171ДБ

## Тематика сайта, название
**Wallpaper Vault** — интерактивная галерея обоев для рабочего стола. Идея реализовать современный веб-каталог высококачественных изображений с удобной навигацией по тематикам и мгновенным предпросмотром в полноэкранном режиме.

## Цель и задачи сайта
Обеспечение быстрого и удобного доступа к графическому контенту для персонализации рабочего пространства.
*   реализация динамической фильтрации контента по категориям (Минимализм, Космос, Природа);
*   верстка адаптивной сетки для корректного отображения превью на любых устройствах;
*   разработка интерфейса модального окна для просмотра оригинальных изображений без ухода с текущей страницы.

## Целевая аудитория
Пользователи персональных компьютеров, дизайнеры и геймеры, которые ищут качественный визуальный контент. Целевая аудитория отличается вниманием к деталям интерфейса, что требует использования минималистичного стиля и высокой скорости работы сайта.

## Содержание сайта
*   шапка сайта с кнопками переключения категорий;
*   основная галерея с адаптивной сеткой карточек изображений;
*   модальное окно для детального просмотра выбранных обоев;
*   подвал сайта с информацией об источниках и авторе.

## Функциональные возможности
*   мгновенная фильтрация изображений по тематическим тегам;
*   просмотр изображений в полном разрешении по клику;
*   анимированные эффекты при взаимодействии с элементами интерфейса;
*   полная адаптивность сайта под мобильные и десктопные разрешения.
